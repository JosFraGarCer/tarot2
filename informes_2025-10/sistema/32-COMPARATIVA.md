## 2. vs Dungeons & Dragons 5e

### Comparativa Directa

| Aspecto | D&D 5e | Proyecto Tarot |
|---------|--------|----------------|
| **Dados** | 1d20 + mods | 2d12 (Habilidad + Destino) |
| **Atributos** | 6 (Fue, Des, Con, Int, Sab, Car) | 9 Facetas en 3 Arcanos |
| **Clases** | 13 clases cerradas | Ocupaciones modulares |
| **Nivel** | 1-20 | 4 Sellos (Iniciado→Leyenda) |
| **Combate** | Táctico con grid | Narrativo con posicionamiento |
| **Magia** | Slots por nivel | Fatiga + Rituales |
| **Licencia** | OGL/ORC | CC BY-SA 4.0 |

### Ventajas de Tarot sobre D&D

| Área | Por qué Tarot es mejor |
|------|----------------------|
| **Narrativa** | El Dado de Destino añade capa narrativa a cada tirada |
| **Personalización** | 5 Cartas permiten más combinaciones que 13 clases |
| **Velocidad** | Combates más cortos (15 vs 45+ minutos) |
| **Modularidad** | Funciona en cualquier género sin hackear |
| **Aprendizaje** | Menos reglas que memorizar |

### Ventajas de D&D sobre Tarot

| Área | Por qué D&D es mejor |
|------|---------------------|
| **Comunidad** | Millones de jugadores, infinito contenido |
| **Herramientas** | VTTs, apps, soporte digital masivo |
| **Contenido** | Miles de manuales, aventuras, settings |
| **Reconocimiento** | "¿Jugamos rol?" = D&D para muchos |
| **Combate Táctico** | Si te gusta el grid, D&D lo hace mejor |

### Público Objetivo

> **D&D:** Jugadores que disfrutan combate táctico y progresión mecánica.
> 
> **Tarot:** Jugadores que prefieren narrativa con estructura, sin grid.

## 3. vs Powered by the Apocalypse (PbtA)

### Comparativa Directa

| Aspecto | PbtA (ej: Dungeon World) | Proyecto Tarot |
|---------|--------------------------|----------------|
| **Dados** | 2d6 + stat | 2d12 + Faceta |
| **Resultados** | 3 niveles (6-, 7-9, 10+) | 4 cuadrantes + Giro |
| **Movimientos** | Ficticios, triggers | Tirada universal |
| **Personaje** | Playbooks | 5 Cartas |
| **DJ** | Sin tirar, solo narrar | Tira para NPCs |
| **Filosofía** | Ficción primero | Ficción con estructura |

### Ventajas de Tarot sobre PbtA

| Área | Por qué Tarot es mejor |
|------|----------------------|
| **Estructura** | Más guía para DJs nuevos |
| **Combate** | Sistema de combate definido (PA, Daño, Protección) |
| **Progresión** | Sellos de Poder dan sensación de avance clara |
| **Consistencia** | Una mecánica universal vs movimientos específicos |

### Ventajas de PbtA sobre Tarot

| Área | Por qué PbtA es mejor |
|------|----------------------|
| **Minimalismo** | Menos reglas, más improvisación |
| **DJ sin tirar** | Flujo narrativo más rápido |
| **Playbooks** | Personajes listos en 5 minutos |
| **Triggers** | Los movimientos guían la ficción |

### Público Objetivo

> **PbtA:** Narradores que quieren mínimas reglas y máxima ficción.
> 
> **Tarot:** Narradores que quieren estructura sin perder narrativa.

## 4. vs Daggerheart

### Comparativa Directa

| Aspecto | Daggerheart | Proyecto Tarot |
|---------|-------------|----------------|
| **Dados** | 2d12 (Hope + Fear) | 2d12 (Habilidad + Destino) |
| **Recurso** | Acumulación Hope/Fear | Efecto inmediato |
| **Personaje** | Ancestries + Classes | 5 Cartas modulares |
| **Combate** | Action tokens | Turnos clásicos |
| **Daño** | Stress + HP | Aguante (PA) |
| **Magia** | Spell slots | Fatiga |

### La Diferencia Clave: Hope/Fear vs Giro Tarot

```
DAGGERHEART:
- Hope (d12 alto) → Acumula tokens → Gasta para beneficio
- Fear (d12 alto) → DJ acumula → Usa para complicar
- Es un sistema de ECONOMÍA DE RECURSOS

PROYECTO TAROT:
- Habilidad > Destino → Efecto INMEDIATO positivo
- Destino > Habilidad → Efecto INMEDIATO negativo
- Dados iguales → GIRO DEL DESTINO (evento único)
- Es un sistema de NARRATIVA INMEDIATA
```

### Ventajas de Tarot sobre Daggerheart

| Área | Por qué Tarot es mejor |
|------|----------------------|
| **Simplicidad** | Sin tracking de tokens |
| **Inmediatez** | Cada tirada tiene efecto completo |
| **Modularidad** | Funciona en cualquier género |
| **Giro del Destino** | Momento único vs acumulación |

### Ventajas de Daggerheart sobre Tarot

| Área | Por qué Daggerheart es mejor |
|------|----------------------------|
| **Tensión acumulada** | Fear creciente = clímax |
| **Producción** | Critical Role + Darrington Press |
| **Comunidad** | Fanbase de CR |
| **VTT integrado** | Diseñado para online |

### Público Objetivo

> **Daggerheart:** Fans de Critical Role, jugadores online.
> 
> **Tarot:** Jugadores que prefieren efectos inmediatos sin tracking.

## 8. Nicho de Mercado

### ¿Para Quién es Proyecto Tarot?

```
┌─────────────────────────────────────────────────────────────┐
│                                                              │
│   TAROT ES PARA TI SI:                                      │
│                                                              │
│   ✅ Te gusta la narrativa pero quieres estructura          │
│   ✅ D&D te parece demasiado pesado                         │
│   ✅ PbtA te parece demasiado ligero                        │
│   ✅ Quieres un sistema que funcione en cualquier género    │
│   ✅ Valoras la personalización del personaje               │
│   ✅ Te gusta que cada tirada cuente una historia           │
│                                                              │
│   TAROT NO ES PARA TI SI:                                   │
│                                                              │
│   ❌ Necesitas combate táctico con miniaturas               │
│   ❌ Prefieres cero reglas (mejor FATE o FKR)               │
│   ❌ Quieres acceso a miles de suplementos ya hechos        │
│   ❌ Tu grupo solo juega D&D y no quiere cambiar            │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### Propuesta de Valor Única

> **"El sistema narrativo que no te deja colgado."**

1. **Estructura sin rigidez:** Tienes reglas claras pero flexibles.
2. **Narrativa mecánica:** El Dado de Destino no es "narrativa del DJ", es parte del sistema.
3. **Universal de verdad:** Las 5 Cartas funcionan en Roma, Star Wars, o Cthulhu.
4. **Personajes únicos:** No hay dos personajes iguales, jamás.
5. **Open Source:** Puedes hacer lo que quieras con el sistema.

### Competidores Directos

| Sistema | Por qué compiten | Por qué Tarot gana |
|---------|------------------|-------------------|
| **Daggerheart** | Mismo concepto de 2d12 | Efectos inmediatos vs acumulación |
| **Genesys** | Sistema universal | Dados especiales vs d12 estándar |
| **Cortex Prime** | Modular | Más simple, menos configuración |
| **Cypher System** | Narrativo universal | Personalización más profunda |
