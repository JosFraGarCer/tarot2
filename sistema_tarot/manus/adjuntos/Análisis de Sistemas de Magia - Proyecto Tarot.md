## 1. Análisis de Universos de Referencia

### 1.1. La Rueda del Tiempo - Características Clave

**Filosofía Mágica:**
- **Fuente Única:** El Poder Único (saidin/saidar) es la fuente de toda magia
- **Especialización Natural:** Cada canalizador tiene afinidades innatas
- **Coste Físico:** Canalizar agota al usuario, puede ser peligroso
- **Aprendizaje Formal:** Requiere entrenamiento estructurado (Torre Blanca, Asha'man)
- **Limitaciones Inherentes:** Fuerza máxima determinada al nacer

**Mecánicas Distintivas:**
- **Tejer:** Combinar hilos de los Cinco Poderes
- **Angreal:** Objetos que amplifican el poder
- **Círculos:** Cooperación entre canalizadores
- **Fatiga:** Uso excesivo causa agotamiento o muerte

### 1.2. Harry Potter - Características Clave

**Filosofía Mágica:**
- **Hechizos Específicos:** Cada efecto requiere un hechizo particular
- **Aprendizaje Académico:** Sistema educativo formal (Hogwarts)
- **Versatilidad:** Amplio rango de efectos posibles
- **Componentes:** Varitas, palabras, gestos, ingredientes
- **Progresión por Estudio:** Mejora a través de práctica y conocimiento

**Mecánicas Distintivas:**
- **Hechizos Catalogados:** Lista específica de efectos conocidos
- **Niveles de Dificultad:** Desde básico hasta EXTASIS
- **Especialización por Materia:** Transfiguración, Encantamientos, etc.
- **Objetos Mágicos:** Varitas, pociones, artefactos

## 2. Aproximaciones de Diseño

### 2.1. **Aproximación A: Magia como Talentos Expandidos**

**Concepto:** Tratar la magia como una extensión del sistema de Talentos de Armas.

**Mecánica:**
- **Talentos Mágicos:** Fuego, Agua, Aire, Tierra, Espíritu (Rueda del Tiempo)
- **Distribución:** Misma que armas (Luchador 3, Mixto 2, No Luchador 1)
- **Tiradas:** Faceta + Talento Mágico vs Dificultad
- **Daño/Efectos:** Similar a armas, pero con efectos especiales

**Aplicación en Rueda del Tiempo:**
```
Aes Sedai (Luchador Mágico): 3 puntos en Cinco Poderes
- Aire 2, Agua 1, Fuego 0, Tierra 0, Espíritu 0
- Tirada: Alma + Talento Aire vs Dificultad
- Efecto: Tejer Aire para mover objetos, crear barreras
```

**Aplicación en Harry Potter:**
```
Mago Entrenado (Mixto): 2 puntos en Escuelas
- Encantamientos 1, Transfiguración 1
- Tirada: Ingenio + Encantamientos vs Dificultad
- Efecto: Lanzar hechizos específicos de esa escuela
```

**Ventajas:**
- Integración perfecta con sistema existente
- Balanceado automáticamente
- Fácil de entender y usar

**Desventajas:**
- Puede sentirse limitado para magia compleja
- No captura la versatilidad de algunos sistemas mágicos

### 2.2. **Aproximación B: Magia con Coste de Recursos**

**Concepto:** La magia consume un recurso específico (Puntos de Poder, Fatiga Mágica, etc.).

**Mecánica:**
- **Puntos de Poder:** Calculados como Alma + Voluntad + Modificadores
- **Coste por Hechizo:** Según potencia y complejidad
- **Recuperación:** Descanso, meditación, o tiempo
- **Sobreesfuerzo:** Gastar más PP del disponible causa daño

**Aplicación en Rueda del Tiempo:**
```
Canalizador:
- Puntos de Poder = Alma × 2 + Voluntad + Fuerza en el Poder
- Tejer Básico: 1 PP
- Tejer Complejo: 3-5 PP
- Sobreesfuerzo: Puedes gastar PA como PP (peligroso)
```

**Aplicación en Harry Potter:**
```
Mago:
- Puntos de Poder = Alma + Voluntad + Nivel de Entrenamiento
- Hechizo Básico: 1 PP
- Hechizo Avanzado: 2-3 PP
- Hechizo EXTASIS: 4-5 PP
```

**Ventajas:**
- Captura el coste y fatiga de la magia
- Permite efectos muy poderosos con limitaciones
- Decisiones tácticas sobre uso de recursos

**Desventajas:**
- Añade complejidad al sistema
- Requiere tracking adicional
- Puede ralentizar el juego

### 2.3. **Aproximación C: Magia por Niveles de Poder**

**Concepto:** Hechizos organizados en niveles, acceso determinado por entrenamiento.

**Mecánica:**
- **Niveles de Hechizo:** 1-5, correspondientes a Sellos
- **Acceso:** Determinado por Sello actual y especialización
- **Tiradas:** Faceta + Especialización vs Dificultad del Nivel
- **Efectos Escalados:** Más poderosos en niveles superiores

**Aplicación en Rueda del Tiempo:**
```
Niveles de Tejido:
- Nivel 1: Efectos básicos (luz, calor menor)
- Nivel 2: Efectos útiles (curación menor, escudo)
- Nivel 3: Efectos poderosos (rayo, curación mayor)
- Nivel 4: Efectos devastadores (balefire menor)
- Nivel 5: Efectos legendarios (balefire, Viaje)
```

**Aplicación en Harry Potter:**
```
Niveles de Hechizo:
- Nivel 1: Primer año (Lumos, Alohomora)
- Nivel 2: Segundo-Tercer año (Expelliarmus, Protego)
- Nivel 3: Cuarto-Quinto año (Patronus, Imperius)
- Nivel 4: Sexto-Séptimo año (Sectumsempra)
- Nivel 5: Maestría (Horrocruxes, Magia Antigua)
```

**Ventajas:**
- Progresión clara y satisfactoria
- Fácil balancear por nivel
- Familiar para jugadores de otros sistemas

**Desventajas:**
- Puede sentirse rígido
- Menos integrado con el núcleo del sistema
- Requiere listas extensas de hechizos

### 2.4. **Aproximación D: Magia Híbrida (Talentos + Potencias)**

**Concepto:** Combinar Talentos mágicos con el sistema de Potencias para efectos únicos.

**Mecánica:**
- **Talentos Base:** Determinan competencia básica
- **Potencia Mágica:** Fuente de poder y efectos especiales
- **Sinergia:** Talentos + Potencia = efectos únicos
- **Intervenciones Mágicas:** Gastar Devoción para efectos poderosos

**Aplicación en Rueda del Tiempo:**
```
Aes Sedai:
- Talentos: Cinco Poderes (como Aproximación A)
- Potencia: "El Poder Único" o "La Torre Blanca"
- Intervención Menor: Repetir dado en tejido crítico
- Intervención Mayor: Tejer sin fatiga, efectos imposibles
```

**Aplicación en Harry Potter:**
```
Mago de Hogwarts:
- Talentos: Escuelas de Magia
- Potencia: "La Magia" o "Hogwarts"
- Intervención Menor: Repetir dado en hechizo difícil
- Intervención Mayor: Hechizo sin varita, magia sin palabras
```

**Ventajas:**
- Máxima integración con sistema existente
- Captura la naturaleza especial de la magia
- Permite efectos narrativos únicos

**Desventajas:**
- Más complejo de balancear
- Puede hacer la magia demasiado poderosa

## 4. Propuesta de Prototipo

### 4.1. **Sistema Híbrido Simplificado**

**Combinar lo mejor de cada aproximación:**

**Base:** Talentos Mágicos (como armas)
- Distribución estándar por arquetipo
- Tiradas: Faceta + Talento vs Dificultad

**Coste:** Puntos de Poder opcionales
- Solo para efectos muy poderosos
- PP = Alma + Voluntad (simple)

**Potencias:** Integración narrativa
- Fuente de poder mágico
- Intervenciones para efectos únicos

**Niveles:** Por Sello
- Acceso a efectos más poderosos por Sello
- Dificultades escaladas apropiadamente

### 4.2. **Ejemplo de Implementación**

**Aes Sedai (Rueda del Tiempo):**
```
Talentos Mágicos: Aire 2, Agua 1, Espíritu 1
Potencia: "El Poder Único"
Puntos de Poder: 6 (Alma 2 + Voluntad 3 + Entrenamiento 1)

Tejer Básico:
- Tirada: Alma + Talento vs Dificultad
- Sin coste de PP para efectos menores

Tejer Poderoso:
- Coste: 1-3 PP según efecto
- Riesgo: Sobreesfuerzo causa daño

Intervención Mayor:
- Coste: 3 Devoción
- Efecto: Tejer imposible, ignorar fatiga
```

**Estudiante de Hogwarts:**
```
Talentos Mágicos: Encantamientos 1, Transfiguración 1
Potencia: "La Magia de Hogwarts"
Puntos de Poder: 4 (Alma 1 + Voluntad 2 + Entrenamiento 1)

Hechizo Básico:
- Tirada: Ingenio + Talento vs Dificultad
- Sin coste para hechizos de primer año

Hechizo Avanzado:
- Coste: 1-2 PP según nivel
- Requiere Sello apropiado

Intervención Menor:
- Coste: 1 Devoción
- Efecto: Repetir dado, hechizo sin varita
```
