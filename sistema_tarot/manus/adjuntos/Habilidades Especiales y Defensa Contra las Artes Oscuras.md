# Habilidades Especiales y Defensa Contra las Artes Oscuras

## Habilidades Especiales Únicas

### Filosofía de Diseño

Las habilidades como **Animago** y **Parsel** son fundamentalmente diferentes de las escuelas de magia normales. Son talentos únicos, a menudo innatos o extremadamente raros, que merecen tratamiento especial.

### Propuesta: Habilidades como Cartas de Trasfondo

**Animago**
- **Tipo:** Carta de Trasfondo
- **Descripción:** El personaje completó la transformación animaga (legal o ilegalmente)
- **Aporte:** Puede transformarse en un animal específico. La transformación es automática una vez aprendida, pero detectar la forma animal requiere Alma + Transfiguración

**Parsel**
- **Tipo:** Carta de Linaje o Trasfondo
- **Descripción:** Capacidad innata (Linaje) o adquirida (Trasfondo) de hablar con serpientes
- **Aporte:** Comunicación automática con serpientes, bonificación para tratar con criaturas reptilianas

**Metamorfomago**
- **Tipo:** Carta de Linaje
- **Descripción:** Habilidad innata de cambiar la apariencia física
- **Aporte:** Cambio de apariencia automático, no requiere varita ni hechizos

**Vidente**
- **Tipo:** Carta de Linaje o Trasfondo
- **Descripción:** Don profético genuino (muy raro)
- **Aporte:** Visiones espontáneas del futuro, bonificación extrema a Adivinación real

### Alternativa: Habilidades Especiales Separadas

Si prefieres tratarlas como habilidades independientes:

| Habilidad | Faceta Base | Descripción |
| :--- | :--- | :--- |
| **Animago** | Alma | Transformación en forma animal específica |
| **Parsel** | Alma | Comunicación con serpientes |
| **Legilimancia** | Alma | Lectura de mentes |
| **Oclumancia** | Voluntad | Protección mental |

## Defensa Contra las Artes Oscuras: El Problema Dual

### El Dilema

Defensa Contra las Artes Oscuras es única porque abarca:
1. **Aspecto Combativo:** Hechizos de combate (Expelliarmus, Stupefy, Protego en duelo)
2. **Aspecto Defensivo:** Protecciones, contrahechizos, detección de magia oscura
3. **Aspecto Académico:** Conocimiento sobre criaturas oscuras, maldiciones, historia

### Propuesta de Consenso: División Funcional

**Para Combate Mágico:**
- Usar el sistema de "Equipo/Conjuro" ya establecido
- Los hechizos defensivos de combate funcionan como armas/ataques

**Para Magia No Combativa:**
- **Defensa Contra las Artes Oscuras** como especialización de Alma
- Cubre: detección de magia oscura, protecciones preventivas, conocimiento académico

### Implementación Detallada

| Situación | Sistema a Usar | Ejemplo |
| :--- | :--- | :--- |
| **Duelo mágico** | Combate (Equipo/Conjuro) | Protego como "bloqueo", Expelliarmus como "ataque" |
| **Detectar maldición** | Alma + Defensa DCAO | Identificar si un objeto está maldito |
| **Crear protección** | Alma + Defensa DCAO | Encantar una casa contra intrusos |
| **Conocer criatura** | Erudición + Defensa DCAO | Saber las debilidades de un Dementor |
| **Contrahechizo** | Alma + Defensa DCAO | Deshacer una maldición no combativa |

### Ejemplos Prácticos

**Harry detectando un Horrocrux:**
- **Habilidad:** Alma (2) + Defensa DCAO (1) = +3
- **Dificultad:** Heroico (15) - los Horrocruxes están muy ocultos
- **Modificadores:** Varita (+2), experiencia previa (+1)
- **Resultado:** Necesita mucha suerte o ayuda

**Hermione creando protecciones en la tienda:**
- **Habilidad:** Alma (3) + Defensa DCAO (2) = +5  
- **Dificultad:** Difícil (12) - protecciones complejas
- **Modificadores:** Varita (+2), tiempo ilimitado (+1)
- **Resultado:** Muy probable éxito

**Snape enseñando sobre Inferi:**
- **Habilidad:** Erudición (4) + Defensa DCAO (3) = +7
- **Dificultad:** Moderado (9) - conocimiento académico
- **Resultado:** Automático para un experto

## Habilidades Mentales Especiales

### Legilimancia y Oclumancia

Estas merecen consideración especial porque son:
- Extremadamente raras
- Requieren entrenamiento específico
- Tienen implicaciones narrativas importantes

**Propuesta:**

**Legilimancia**
- **Faceta Base:** Alma (para la conexión mágica)
- **Especialización:** Legilimancia como Umbral de Competencia
- **Limitaciones:** Requiere contacto visual, puede ser bloqueada por Oclumancia

**Oclumancia**  
- **Faceta Base:** Voluntad (para la resistencia mental)
- **Especialización:** Oclumancia como Umbral de Competencia
- **Función:** Defensa automática contra Legilimancia del mismo nivel o inferior

### Mecánica de Confrontación Mental

**Legilimante vs Oclusor:**
- Tirada enfrentada: Alma + Legilimancia vs Voluntad + Oclumancia
- El ganador controla el flujo de información
- Empate = información parcial o confusa

## Integración con el Sistema Base

### Como Cartas de Trasfondo

Las habilidades especiales más raras funcionan mejor como Cartas de Trasfondo:

**"Entrenamiento Animago"**
- Completaste el proceso de transformación animaga
- Puedes transformarte en tu forma animal específica
- La transformación es automática, pero mantener el control bajo estrés requiere tiradas

**"Don Profético"**
- Posees el verdadero don de la adivinación
- Recibes visiones espontáneas (controladas por el DJ)
- Bonificación extrema (+3) a Adivinación cuando el don se activa

### Como Especializaciones Raras

Para habilidades que requieren entrenamiento:

**Legilimancia/Oclumancia**
- Especializaciones de Alma/Voluntad respectivamente
- Requieren maestro para aprender
- Muy raras, solo disponibles con justificación narrativa

## Recomendación Final

**Para Defensa Contra las Artes Oscuras:**
- **Combate:** Usar sistema Equipo/Conjuro
- **No Combate:** Especialización de Alma + modificadores según situación
- **Académico:** Erudición + Defensa DCAO para conocimiento teórico

**Para Habilidades Especiales:**
- **Innatas/Raras:** Cartas de Linaje o Trasfondo
- **Entrenables:** Especializaciones muy raras de Alma/Voluntad
- **Únicas:** Tratamiento caso por caso según la narrativa

Esto mantiene la simplicidad del sistema mientras da el peso narrativo apropiado a estas habilidades especiales.
