# Perfil de Personaje: Luchador de Sello de Iniciado

Este documento define el perfil de personaje que se utilizará para las simulaciones de combate en el sistema "Proyecto Tarot". Se creará un personaje jugador (PJ) y un personaje no jugador (PNJ) con características idénticas para asegurar un entorno de prueba equilibrado.

## 1. Atributos (Facetas)

La distribución de puntos se basa en un personaje de Sello de Iniciado, con un límite de 3 puntos por Faceta en la creación.

- **Arcano Primario (Físico):** 5 puntos
- **Arcano Secundario (Mental):** 3 puntos
- **Arcano Terciario (Espiritual):** 1 punto

| Arcano | Faceta | Valor | Nivel de Competencia |
| :--- | :--- | :--- | :--- |
| **Físico** | Fuerza | 3 | Excepcional |
| | Agilidad | 1 | Entrenado |
| | Vigor | 1 | Entrenado |
| **Mental** | Percepción | 2 | Competente |
| | Ingenio | 1 | Entrenado |
| | Erudición | 0 | Sin Destacar |
| **Espiritual** | Voluntad | 1 | Entrenado |
| | Carisma | 0 | Sin Destacar |
| | Alma | 0 | Sin Destacar |

## 2. Capacidades de Combate

Estas son las estadísticas derivadas que se utilizarán directamente en la simulación de combate, basadas en las reglas de la versión 0.1.1.0 del documento de diseño.

| Característica | Valor | Descripción |
| :--- | :--- | :--- |
| **Puntos de Aguante** | **11** | Base de Ocupación (10, asumido) + Vigor (1) |
| **Tirada de Ataque** | 1d12 + 3 | Dado de Habilidad (d12) + Faceta de Fuerza (3) |
| **Dificultad de Ataque** | 12 | Dificultad "Difícil" para un combate activo |
| **Arma** | Espada | Daño base de 3 |
| **Protección** | Cota de Malla | Reducción de daño de 2 |
| **Daño por Golpe** | **1** | Daño del Arma (3) - Protección (2) = 1 |

## 3. Probabilidad de Impacto

Para impactar, la tirada de ataque (1d12 + 3) debe igualar o superar la dificultad de 12. Esto requiere un resultado de **9 o más** en el dado de 12 caras.

- **Probabilidad de Éxito:** 4/12 = **33.3%**
