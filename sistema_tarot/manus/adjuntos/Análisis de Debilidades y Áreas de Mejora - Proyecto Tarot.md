# Análisis de Debilidades y Áreas de Mejora - Proyecto Tarot

## Debilidades Identificadas

### 1. **Inconsistencias en la Distribución de Puntos de Creación**

El documento presenta **discrepancias significativas** entre diferentes secciones respecto a la asignación de puntos durante la creación de personajes. En la sección 2.3 se menciona una distribución de **5/3/1 puntos**, mientras que el conocimiento base del sistema indica **6/3/1 puntos**. Esta inconsistencia puede generar confusión durante la implementación y requiere clarificación inmediata.

La **Regla Opcional del Sacrificio del Arcano** también presenta problemas matemáticos. Con la distribución actual, la "Opción B" para alcanzar la trascendencia no es viable durante la creación inicial, lo que sugiere que esta regla fue diseñada para un sistema de puntos diferente y no se actualizó correctamente.

### 2. **Falta de Ejemplos Concretos de Cartas**

Aunque el sistema de cinco cartas fundamentales es conceptualmente sólido, el documento **carece de ejemplos específicos** que ilustren cómo funcionan en la práctica. Los jugadores y directores de juego necesitan ver cartas completas de Linaje, Entorno, Trasfondo, Ocupación y Potencia para comprender completamente el alcance y poder de cada una.

Esta ausencia de ejemplos concretos dificulta la evaluación del equilibrio entre diferentes opciones y puede resultar en interpretaciones inconsistentes entre diferentes grupos de juego.

### 3. **Sistema de Combate Subdesarrollado**

El sistema de combate, aunque funcional, presenta varias **limitaciones significativas**:

La **iniciativa simplificada** ("los PJ actúan primero por defecto") puede resultar demasiado favorable para los jugadores y reducir la tensión táctica. Aunque existe una excepción para emboscadas, no se abordan situaciones más complejas como combates múltiples o enfrentamientos entre grupos mixtos.

El **sistema de daño** es excesivamente simplista. La reducción punto por punto de la protección, aunque matemáticamente clara, no refleja la realidad de cómo funcionan diferentes tipos de armadura contra diferentes armas. Un sistema más sofisticado podría considerar la efectividad relativa de armas perforantes versus contundentes contra diferentes tipos de protección.

### 4. **Ambigüedad en el Sistema del Destino**

Aunque el **Dado de Destino** es conceptualmente innovador, su implementación práctica presenta desafíos. El documento no proporciona **suficientes ejemplos específicos** de cómo interpretar los diferentes rangos de destino en situaciones concretas.

La **Escala del Destino** mencionada en el conocimiento base (con diferencias calculadas entre dados) no aparece en el documento principal, creando una **desconexión entre la documentación** que puede confundir a los usuarios.

### 5. **Sistema de Progresión Limitado**

Los **Sellos de Poder** proporcionan una estructura clara, pero el sistema de progresión dentro de cada Sello es **relativamente rígido**. El coste de mejora basado en el "Nuevo Valor" puede resultar prohibitivamente caro para mejoras altas, potencialmente estancando el desarrollo del personaje.

La progresión se centra principalmente en **mejoras numéricas** (subir Facetas) sin suficientes opciones para el crecimiento horizontal o la adquisición de nuevas capacidades cualitativas.

### 6. **Gestión de Recursos Poco Desarrollada**

El sistema de **Puntos de Devoción** es interesante pero limitado. Con solo 3 puntos máximos y costes de 1 y 3 puntos para las intervenciones, los jugadores tienen muy pocas oportunidades de usar estas mecánicas en una sesión típica.

No existe un **sistema de recursos secundarios** (como puntos de fatiga, estrés mental, o recursos materiales) que podría añadir profundidad táctica y narrativa al juego.

### 7. **Falta de Orientación para el Director de Juego**

El documento se centra principalmente en las **mecánicas del jugador** pero proporciona poca orientación específica para los directores de juego. Faltan secciones sobre:

- Cómo establecer dificultades apropiadas para diferentes situaciones
- Guías para interpretar y narrar los resultados del Dado de Destino
- Consejos para equilibrar encuentros y desafíos
- Herramientas para la creación de PNJ usando el mismo sistema

### 8. **Terminología Inconsistente**

El documento utiliza términos como **"Facetas"** y **"Palos"** de manera intercambiable, lo que puede generar confusión. La terminología del tarot (Arcanos, Palos) es temáticamente apropiada pero puede resultar confusa para jugadores no familiarizados con estos conceptos.

### 9. **Ausencia de Mecánicas Sociales Específicas**

Aunque el sistema incluye atributos sociales (Carisma, Voluntad), no proporciona **mecánicas específicas** para la resolución de conflictos sociales, negociaciones, o interacciones complejas entre personajes.

### 10. **Limitaciones del Sistema Universal**

Paradójicamente, la **ambición universal** del sistema puede ser también una debilidad. Al intentar funcionar en cualquier ambientación, algunas mecánicas pueden resultar genéricas o inadecuadas para géneros específicos que requieren reglas especializadas.

## Áreas de Mejora Prioritarias

### **Inmediatas (Críticas)**
1. **Resolver inconsistencias** en la distribución de puntos de creación
2. **Crear ejemplos completos** de las cinco cartas fundamentales
3. **Clarificar la implementación** del sistema del Dado de Destino

### **Corto Plazo (Importantes)**
1. **Expandir el sistema de combate** con más opciones tácticas
2. **Desarrollar guías específicas** para directores de juego
3. **Estandarizar la terminología** en todo el documento

### **Medio Plazo (Deseables)**
1. **Añadir mecánicas sociales** específicas
2. **Expandir el sistema de progresión** con opciones horizontales
3. **Desarrollar sistemas de recursos** adicionales para mayor profundidad táctica
